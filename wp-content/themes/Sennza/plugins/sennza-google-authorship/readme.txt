"Google Authorship" is a plugin that adds rel="me", rel="author" and a link to the Author's Google Profile.

== Description ==

This plugin has been developed with the intention of making the process of adding [Google Authorship](http://www.google.com/support/webmasters/bin/answer.py?answer=1229920&hl=en "Google Authorship") to your WordPress site as simple as possible.

Inspiration for this plugin has been taken from the following articles from Joost de Valk:
[WordPress](http://wordpress.org/ "Your favorite software")
[rel="author" and rel="me" in WordPress](http://yoast.com/wordpress-rel-author-rel-me/ "rel=author and rel=me in WordPress")
[Highlighting WordPress authors in search](http://yoast.com/highlighting-wordpress-authors-search/ "Highlighting WordPress authors in search")

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `sennza-google-authorship.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Navigate to 'Settings' -> Google Authorship and follow the steps for configuration
